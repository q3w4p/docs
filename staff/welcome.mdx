## Quick Reference

| Action | Command |
| --- | --- |
| Check history | `,history @user` |
| Warn a member | `,warn @user [reason]` |
| Mute / timeout | `,mute @user [duration] [reason]` |
| Unmute | `,unmute @user` |
| Jail a member | `,jail @user [reason]` |
| Unjail | `,unjail @user` |
| Kick | `,kick @user [reason]` |
| Ban | `,ban @user [reason]` |
| Clear messages | `,purge [amount]` |
| Lock a channel | `,lock` |
| Server lockdown | `,lockdown` |

<Info>
  bleed's default prefix is `,` — confirm with a Superior if commands aren't working.
</Info>
