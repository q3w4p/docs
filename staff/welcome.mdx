## Quick Reference

| Action | Command |
|---|---|
| Warn a member | `,warn @user [reason]` |
| Mute/timeout | `,timeout @user [duration] [reason]` |
| Jail a member | `,jail @user [reason]` |
| Unjail | `,unjail @user` |
| Ban | `,ban @user [reason]` |
| Check history | `,history @user` |
| Clear messages | `,purge [amount]` |
| Lock a channel | `,lock` |

<Info>
  bleed's default prefix is `,` — confirm the prefix used in edates with your Head Admin if commands aren't working.
</Info>
