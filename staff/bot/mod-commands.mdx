## Check History

Always run this first.

```bash
,history @user
```

---

## Warn

```bash
,warn @user [reason]
```

```bash
# Example
,warn @user Spamming in #general
```

---

## Timeout / Mute

```bash
,mute @user [duration] [reason]
```

```bash
# Examples
,mute @user 5m Hard-r bypass
,mute @user 1h Repeated extreme racism after warnings
```

To remove a timeout early:

```bash
,unmute @user # use this only if you mistakenly muted a user
```

---

## Jail

Restricts the member to the jail channel only.

```bash
,jail @user [reason]
,jail @user [duration] [reason]
```

```bash
# Examples
,jail @user Soft gore
,jail @user 2d Soft gore
```

To unjail:

```bash
,unjail @user (reason)
```

<Warning>
  Jails may not be ended prematurely unless the member is proven innocent. Only Admin+ may interfere with another staff member's jail case.
</Warning>

## Kick

```bash
,kick @user [reason]
```

---

## Ban

```bash
,ban @user [reason]
,ban @user [duration] [reason]
```

```bash
# Examples
,ban @user Gore # if no duration then its indefinite / permanant
,ban @user 7d Repeated offenses
```

To unban:

```bash
,unban @user (reason)
```

Ban by user ID (even if they've already left):

```bash
,hackban [user_id] [reason] # also you can native just use ,ban
```

## Purge Messages

```bash
,purge [amount]
,purge @user [amount] 
```

<Warning>
  Purges are irreversible. Double-check the amount before running.
</Warning>

## Lock Channel

```bash
,lock (channel) (reason) # only use for raids
,unlock (channel)
```

## Add a Note (Staff-only)

Private notes visible only to staff.

```bash
,note add @user [note]
,note list @user
,note remove @user [note_id]
```
