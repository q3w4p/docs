## What is bleed?

[bleed](https://bleed.bot) is the bot powering moderation, anti-nuke, auto-mod, and more in edates. Every staff member needs to be fluent with its commands.

- Website: [bleed.bot](https://bleed.bot)
- Docs: [docs.bleed.bot](https://docs.bleed.bot)

---

## Prefix

bleed's default prefix is `,` --- confirm with anyone if commands aren't responding.

```bash
,help              # list of available commands
,help [command]    # info + usage for a specific command
```
