## User Info

```bash
,userinfo @user      # account age, join date, roles, badges
,avatar @user        # displays their avatar
,banner @user        # displays their banner
```

---

## Server Info

```bash
,serverinfo          # member count, boosts, creation date, etc.
,roles               # list all roles in the server
```

---

## Role Management

```bash
,role add @user [role]
,role remove @user [role]
```

---

## Slowmode

```bash
,slowmode [seconds]
,slowmode off
```

---

## Lookup

```bash
,lookup [user_id]    # look up a user by ID, even if not in server
```
